# notes-2d-data

## What the dataset contains

- File: data/2d-data.csv
- Rows: 6000
- Columns: 2 (x, y)

## Quick checks I ran

- Missing values: 0
- Data types: float64 for both x and y

## Visualization choice

- Plot type: Hexbin density plot
- Reason: Large number of points, density is clearer than raw scatter
- Axis labels: x and y
